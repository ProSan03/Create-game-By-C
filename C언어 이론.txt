<C언어 이론>
#include <stdio.h> = 스탠다드 인풋 아웃풋 => 키보드와 마우스를 이용해서 입력하는 보편적인 형태 == 없으면 printf, scanf 못씀
거의 모든 함수에서 함수의 끝은 항상 ';'(세미클론) 을 붙인다.

자료형)
정수 - int
실수 - double
문자 - char (1자리) => 문자열은 string(C언어에는 X)

연산자)
- 산술 : '+', '-', '*', '/'(나누기 연산) '%'(나머지 연산)
- 논리 : !(not), ||(OR), &&(AND)
- 관계 : >, <, >=, <=, +=, -=, !=, /=, %= 등 
- 증감 : a++, ++a, a--, --a 
(※참고 : a+=1 ==> a = a+100)

제어문)
- 조건문 : if, switch (차이점: switch는 케이스 1,2를 걸고(안에 break;을 넣음) default;로 마무리 매개변수에 구애 받음
		//  if는 조건을 걸고 각 분기에 따라 달라지는 것 + 매개변수 자유로움)
- 반복문 : for, while, do-while // while -> do-while -> for로 진화함 
-> for : (구조: 초기식(초기화) -> (반복)조건식 -> 내용문 -> 최종식(끝동작) / '초기식;조건식;최종식'으로 표현)
-> while  :매개변수 조건식에 따라 몇번 반복할 지 판단 // 만약 true가 아닌 false라면 while 내용은 다 무시함.
-> do-while : 무조건 한번 실행한 후(do 부분) while문으로 바뀜 // 즉, 조건 검사를 while은 제일 처음에, do-while은 do 실행 이후에
